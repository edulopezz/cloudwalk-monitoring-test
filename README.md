# CloudWalk Monitoring Analyst Test

This repository contains my submission for the CloudWalk Monitoring Analyst Test.

## Contents

report/
Technical analysis document explaining the anomaly detection process.

data/
Original datasets used in the challenge.

sql/
Queries used to analyze checkout behavior and transaction monitoring metrics.

python/
Example implementation of a monitoring endpoint for anomaly detection.

## Approach

The solution focuses on:

- historical baseline comparison
- SQL-based aggregation
- statistical analysis of transaction rates
- rule-based anomaly detection
- monitoring endpoint example in Python

Charts and exploratory analysis were performed using Google Sheets.
